### 🏠 [Homepage](https://github.com/kongxiangyiren/vite_cli#readme)

## Prerequisites

- node ^14.18.0 || >=16.0.0

## Install

```sh
npm install @feiyuhao/vite_cli --global
```

## Usage

### 创建项目

```sh
vc create
```

### 创建自动化

```sh
vc deploy
```

### 查看版本

```sh
vc -v
```

![](create.gif)

## UnInstall

````sh
npm uninstall @feiyuhao/vite_cli --global

````
